== Description ==

# Cerrar Web #
Lightweight Wordpress plugin that if active will "close" a website. Logged in users will be able navigate the site normaly.

== Installation ==

1. Upload `Cerrar-Web` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
